# IDLinens RAG Assistant

An intelligent assistant based on Retrieval-Augmented Generation (RAG) designed to answer questions about the IDLinens HA platform using official documentation.

## Project Overview

This project was developed as the final challenge for the Alura AI Agents course.

The assistant uses Retrieval-Augmented Generation (RAG) to search through the official documentation of the IDLinens HA platform and generate accurate answers based on the retrieved information.

The knowledge base is composed of technical and user documentation created specifically for the project.

## Main Features

- Answer questions about the Android application.
- Answer questions about the Dashboard.
- Search information using semantic similarity.
- Cite information from the official documentation.
- Reduce the time required to locate operational procedures.

## Technologies

- Python
- LangChain
- OpenAI
- FAISS
- Streamlit
- Oracle Cloud Infrastructure (OCI)

## Documentation

The assistant uses the following documentation:

- Android User Manual
